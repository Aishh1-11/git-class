 Command for listing git config
  >git config --list :

 command for adding name
  >git config --global user.name "user_name"

 command for adding email
  >git config --global user.email "email@gmail.com"

 command for initializing git repository
  > git init

 command for listing all untracked files
  >git status

 command for add all untracked files to staging area
  >git add -A

 command for adding all files to repository
  >git commit -m "commit message"
